# Books Store App

Books Store

## Live demo

Official website: https://books-shoop.vercel.app

## Main technology used

- ReactJS, Typescript, TailwindCSS
- Redux-Toolkit
- Swiper
- React-Icons
- React-Router-Dom
- Hooks
- Scss

## Features

- Well-designed homepage
- Search by name.
- Books detail
- detail pages and search
- responsive design
